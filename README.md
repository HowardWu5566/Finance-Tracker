## 使用說明

1. Clone 專案

```
git clone https://github.com/HowardWu5566/Finance-Tracker.git
```

2. 進入資料夾

```
cd Finance-Tracker
```

3. 安裝套件

```
pip install requests flask matplotlib
```

4. 資料庫初始化

```
python db_setting.py
```

5. 啟動伺服器

```
python index.py
```

6. 在瀏覽器網址列輸入 (port 預設為 5000)
```
127.0.0.1:{port}
```
